# Speeches: 2028 Dem Prez Primary

Major speeches from the prospective 2028 Democratic presidential field, packaged as a podcast. Part of the Speech Pods network.

- **Website:** https://speechpods.github.io/speeches-2028/
- **RSS feed:** https://speechpods.github.io/speeches-2028/feed.xml

Each episode contains the full audio of one notable recent speech by a prospective candidate, with a short narrated introduction setting the scene (who, what occasion, where, when) and a closing recap over the fading applause.

Audio is drawn from publicly posted recordings; each episode links its original source video. This is an independent, unofficial archive, not affiliated with any campaign or committee.

Built with `yt-dlp`, `ffmpeg`, ElevenLabs narration, and Python.
